# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета "Daily Gym Statistic" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:**  Данные представляют собой ежедневные записи посещений фитнес-клуба за 2024 год (2600 транзакций). Содержат информацию о клиентах (ID, возраст, пол, тип абонемента), тренировках (тип, длительность, калории), времени посещения и статусе явки.
**Цель лабораторной:** аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения  
- `data/` — датасет или скрипт загрузки
